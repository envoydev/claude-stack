Cursor stack inventory

The Cursor counterpart of claude-stack.html. Provisioned by cursor-stack.{sh,ps1} into a project's self-contained .cursor/ tree.
Cursor has no /plugin install system. Capabilities arrive three ways: MCP servers (native, via .cursor/mcp.json), native features (Skills, Bugbot, Rules), and Open VSX VS Code extensions. Claude marketplace plugins themselves cannot run in Cursor - only the MCP servers bundled inside them are portable. One exception worth adding manually: superpowers ships its own Cursor plugin - run /add-plugin superpowers in Cursor chat (this script can't script that UI step). The skills below run as Cursor Skills (agentskills.io standard, copied into .cursor/skills/); MCP servers work identically to Claude. See the Claude plugins → Cursor equivalent table for how each Claude plugin maps.
