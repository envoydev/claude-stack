Editing web / Angular / Ionic frontend code - load the governing house-style skill before the edit; conventions are the source of truth, not recall. `typescript` is the baseline for any `.ts`/`.tsx`/`.js`/`.jsx`/`.mjs`/`.cjs` file. `angular-conventions` layers on for Angular component / service / directive / pipe / guard files and templates - an Ionic/Capacitor app shares the same conventions. `angular-styling` governs `.scss`/`.css`. A file that is both an Angular file and TypeScript wants both skills. Skip one-line tweaks.
